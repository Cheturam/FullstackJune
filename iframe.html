We will use a div tag.
This a div tag
Div tag allows us to write text from new line & after the text is executed the next execution will be from the new line.
Div tag is Block Level Element.
Now we will use a Span tag. This is a Inline Element. It allows us to add the attributes to text but Break tag does not. Attributes like (style="color: red"). New text will start just after the execution of span tag. New text will not be executed in the new line.
